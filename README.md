# Argos1

Welcome to the Argos1 project. This project uses a monorepo structure with pnpm workspaces to manage multiple packages efficiently.

## Setup

1. Install [pnpm](https://pnpm.io/):
   ```sh
   npm install -g pnpm
   ```
2. Install dependencies:
   ```sh
   pnpm install
   ```
3. Run the project:
   ```sh
   pnpm start
   ```
